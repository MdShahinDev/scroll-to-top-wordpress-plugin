Simple Scroll to top plugin will help you to enable Back to Top button to your WordPress website.

== Description ==
Easily add a customizable "Scroll to Top" button to your WordPress site with Customizable Swift Scroll To Top! This plugin allows users to add a scroll-to-top button to their site and fully customize it to suit their design preferences. You can adjust the icon, colors, and positioning, making it an ideal addition for any website that wants to improve user experience by providing an intuitive, smooth, and stylish way to navigate back to the top of the page.

== Installation and Usage ==

Install the plugin through the WordPress plugin directory or by uploading the files to your server.
Activate the plugin through the 'Plugins' menu in WordPress.
Customize the button settings through customize option "Scroll To Top" menu.

* Go to your Dashboard after installation and navigate to "Appearance > Customize > Scroll To Top" to configure the accordion.
